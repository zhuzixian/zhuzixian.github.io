Nikola intends to let you show code easily via listings:

hello.py (源代码)

#!/usr/bin/python

import sys


def hello(name='world'):
    greeting = "hello " + name
    print(greeting)


if __name__ == "__main__":
    hello(*sys.argv[1:])
